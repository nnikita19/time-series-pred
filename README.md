# time-series-pred
# Objective
To forecast value of the 10th year for parameter 9,10,11,12 and 13 using concept of time series forecasting
# Methodology
Used ARIMA model to predict values for column parameter 9, 10, 11, 12 and 13, using values of the previous nine years as data points for training the model and then predict the value of 10th year.
